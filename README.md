## QuizletCheats
cheats for quizlet

# how to install:
PC: Install the [userscript](https://github.com/SaltSaladd/QuizletCheats/raw/refs/heads/main/quizlet-hacks-gui.user.js) with tampermonkey/violentmonkey or a userscript manager

Android: Install firefox for android, install the violentmonkey extension, then install the [userscript](https://github.com/SaltSaladd/QuizletCheats/raw/refs/heads/main/quizlet-hacks-gui.user.js)

iOS: Install [userscripts](https://apps.apple.com/us/app/userscripts/id1463298887), then install the [userscript](https://github.com/SaltSaladd/QuizletCheats/raw/refs/heads/main/quizlet-hacks-gui.user.js) (i think?? i dont have an ios device to test)
